Feedstock for [fastnlo](https://fastnlo.hepforge.org) — part of [hep-forge](https://anaconda.org/hep-forge).

FastNLO: fast pQCD calculations for hadron-induced processes

## Install

```bash
conda install -c hep-forge -c conda-forge fastnlo
```
